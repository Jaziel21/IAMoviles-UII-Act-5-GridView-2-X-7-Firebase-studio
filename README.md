A new Flutter project created in Firebase Studio.
🚀 Prompt del Proyecto: Librería AJMG
"Actúa como un desarrollador experto en Flutter. Genera el código completo en un único archivo main.dart para una aplicación de catálogo llamada 'LIBRERÍA AJMG', optimizada para ejecutarse en Firebase Studio (Project IDX).

Requerimientos de Interfaz:

AppBar: Color café oscuro (0xFF5D4037), título 'LIBRERÍA AJMG' centrado, en negrita y color blanco. Sin sombras (elevation: 0).

Cuerpo (Grid): Implementa un GridView.builder con un scroll vertical fluido.

Configuración: 2 columnas por 7 filas (total 14 elementos).

Relación de aspecto (childAspectRatio): 0.65 para que las tarjetas sean alargadas y estéticas.

Diseño de Tarjetas (Cards): Cada celda debe ser una Card con elevación 4 y bordes redondeados (12px).

Imagen: Debe ocupar la parte superior de la tarjeta usando Image.network. Usa imágenes de libros o bibliotecas de alta calidad desde Unsplash o Picsum.

Información (3 filas): Debajo de la imagen, incluye:

Fila 1: Título del libro en negrita.

Fila 2: Nombre del autor en color café claro.

Fila 3: Una fila (Row) con 5 iconos de estrellas (Icons.star) en color ámbar.

Restricciones de Navegación:

IMPORTANTE: Elimina cualquier rastro de TabBar, BottomNavigationBar o menús inferiores. El GridView debe ocupar todo el espacio disponible debajo del AppBar.

Usa un ThemeData basado en Colors.brown con useMaterial3: true.

El código debe ser limpio, con nombres de variables en español referentes al negocio de la librería."
